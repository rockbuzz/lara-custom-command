# Lara Custom Command

Laravel Custom Command

## Requirements

PHP: >=7.3

## Install

```bash
$ composer require rockbuzz/lara-custom-command
```

```php
$php artisan cache:renew-all
$php artisan db:show-tables
$php artisan db:show-tables
$php artisan db:describe tableName
$php artisan db:import path name
```
